# RN-offline boilerplate

## Getting started
Install dependencies:  
```yarn install```

Run on iOS:  
```yarn ios```

Run on Android:  
```yarn android```

Run tests:  
```yarn test```

## E2E tests
Install detox:  
https://github.com/wix/Detox

Run E2E iOS tests:  
```yarn e2e:ios```

Run E2E Android tests:  
```yarn e2e:android```
